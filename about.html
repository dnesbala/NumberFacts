ABOUT

This is a simple API based webpage.

It will return a fact for any number you entered.

You can select any one from the three options : Trivia, Maths and Years.

Trivia will return a number fact related to the general knowledge.

Maths will return a number fact associated with any mathematical facts.

Years will return a fact related to the event associated with that particular date or year.
